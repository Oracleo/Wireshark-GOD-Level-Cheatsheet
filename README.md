## рҹҺҜ Interview Scenarios Covered

Real questions you'll be asked, with full methodology answers:

```
1. "Detect a port scan in a PCAP"
   вҶ’ tcp.flags.syn==1 && tcp.flags.ack==0 + Statistics вҶ’ Conversations

2. "Difference between capture and display filters"  
   вҶ’ BPF (kernel-level, pre-capture) vs Wireshark syntax (post-capture, protocol-aware)

3. "Extract credentials from a PCAP"
   вҶ’ http.authbasic + ftp PASS + frame contains + Follow TCP Stream

4. "Investigate suspected malware on a host"
   вҶ’ ip.addr вҶ’ Protocol Hierarchy вҶ’ Conversations вҶ’ IO Graph вҶ’ beaconing analysis

5. "Detect data exfiltration"
   вҶ’ Conversations sorted by bytes + Export Objects + large payload filters

6. "Detect ARP poisoning / MITM"
   вҶ’ arp.duplicate-address-detected (Wireshark auto-flags this)

7. "Investigate a web attack"
   вҶ’ http.request + response codes + URI injection signatures

8. "Capture on a remote server (no GUI)"
   вҶ’ tshark -i eth0 -w /tmp/cap.pcap + SCP + analyze locally

9. "What is DNS tunneling and how do you detect it?"
   вҶ’ Long subdomains, high query frequency, large TXT responses, regex filters

10. "First 3 things you check in an unknown PCAP"
    вҶ’ Expert Information вҶ’ Protocol Hierarchy вҶ’ Conversations sorted by bytes
```
