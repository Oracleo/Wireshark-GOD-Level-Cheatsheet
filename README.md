3. "Extract credentials from a PCAP"
   ‚Üí http.authbasic + ftp PASS + frame contains + Follow TCP Stream

4. "Investigate suspected malware on a host"
   ‚Üí ip.addr ‚Üí Protocol Hierarchy ‚Üí Conversations ‚Üí IO Graph ‚Üí beaconing analysis

   ‚Üí Conversations sorted by bytes + Export Objects + large payload filters

6. "Detect ARP poisoning / MITM"
   ‚Üí arp.duplicate-address-detected (Wireshark auto-flags this)

7. "Investigate a web attack"
   ‚Üí http.request + response codes + URI injection signatures

8. "Capture on a remote server (no GUI)"
   ‚Üí tshark -i eth0 -w /tmp/cap.pcap + SCP + analyze locally

9. "What is DNS tunneling and how do you detect it?"
   ‚Üí Long subdomains, high query frequency, large TXT responses, regex filters

10. "First 3 things you check in an unknown PCAP"
    ‚Üí Expert Information ‚Üí Protocol Hierarchy ‚Üí Conversations sorted by bytes
